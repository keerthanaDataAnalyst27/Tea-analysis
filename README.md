Coffee Shop Sales Dashboard

Overview

The Coffee Shop Sales Dashboard is an interactive data analytics project developed using Python, Streamlit, Pandas, and Plotly. The dashboard provides insights into coffee shop sales performance, customer purchasing patterns, product category trends, and store-wise revenue analysis.

This project helps business owners and analysts make data-driven decisions by visualizing key sales metrics in a simple and user-friendly interface.

---

Features

- Total Revenue Analysis
- Total Orders Tracking
- Average Order Value Calculation
- Revenue Trend Visualization
- Product Category Performance Analysis
- Store Location Revenue Comparison
- Hourly Revenue Analysis
- Top Selling Products Dashboard
- Interactive Charts and Filters
- Responsive Streamlit Interface

---

Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- NumPy

---

Dataset Information

The dataset contains coffee shop transaction records including:

- Transaction ID
- Transaction Date
- Transaction Time
- Product Category
- Product Details
- Store Location
- Quantity Sold
- Unit Price

Revenue is calculated using:

Revenue = Transaction Quantity × Unit Price

Installation

Clone the repository:

git clone https://github.com/your-username/coffee-shop-dashboard.git

Navigate to the project folder:

cd coffee-shop-dashboard

Install required packages:

pip install -r requirements.txt

---

Run the Application

streamlit run dashboard.py

The application will open in your default browser.
